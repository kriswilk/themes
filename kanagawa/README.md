# Kanagawa

Based on [Kanagawa](https://github.com/rebelot/kanagawa.nvim) by [Tommaso Laurenzi](https://github.com/rebelot).

Adapted by [Kris Wilk](https://github.com/kriswilk)

## Notes

- All variants add bright white (#FFFEF9).
- "neutral" variants shift whites toward neutral gray.
- "oled" variants have true black background.
