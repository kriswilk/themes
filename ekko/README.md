# Ekko

Based loosely on the palette from Atom's "One Dark" theme.

Adapted by [Kris Wilk](https://github.com/kriswilk)

## Notes

Every RGB value consists of matching pairs of hex digits. For instance, blue is "#4499DD". This curiosity was deliberate. It also explains the name "Ekko", which evokes repetition ("echo") and contains a matched pair of letters.
